# ProConvert Studio 📄✨

[![Python Version](https://img.shields.io/badge/python-3.7%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

**ProConvert Studio** is a comprehensive PDF manipulation toolkit with a modern GUI. Convert, edit, sign, watermark, and secure your PDF documents with ease!

## ✨ Features

### 🔄 **Universal Converter**
- Images ↔ PDF
- PDF ↔ Word
- PDF ↔ PowerPoint
- PDF → Text/HTML

### ✏️ **PDF Editor**
- Rotate, delete, extract pages
- Thumbnail preview
- Undo/Redo

### 🔀 **PDF Merger**
- Merge multiple PDFs
- Drag-drop reordering

### ✍️ **Digital Signatures**
- Draw and save signatures
- Apply to PDF

### 💧 **Watermark**
- Text and image watermarks
- Customizable position and opacity

### ✂️ **Crop & Resize**
- Visual crop selection
- Auto-crop margins

### 🔒 **Security**
- AES-256 encryption
- PDF decryption

### 🔍 **OCR**
- Extract text from PDF/images
- Multiple language support
